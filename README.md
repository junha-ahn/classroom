# Classroom

Schoool project

Vue.js, ejs, bootstrap, jQuery(ajax)

Node.js, Express


저작권 허가 및 라이선스에 대해, 어기고 있는 사항이 있다면 알려주세요.

## 설치 방법 (Installation)

* ```git clone <repository-url>```

* ```cd classroom```
* ```npm install ```
* ```npm install -g nodemon ```

## 사용 방법 (Running)
~~~~
npm start
~~~~
